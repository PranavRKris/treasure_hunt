Artefact III — The Celestial Codex (Segment 3 · 3-4 digits)

A celestial map reveals three constellations. The ancients believed each constellation's power was determined by a sacred calculation. You must perform the ritual mathematics to unlock the vault's third seal.

THE ASTRONOMER'S FORMULA

"For each constellation, calculate its sacred value:
Value = (Number of Stars × Number of Connections) + Number of Stars"

• Stars: Count all visible points in the constellation
• Connections: Count the lines linking stars together
• Order: Calculate for Constellation I, II, and III (west to east)

CALCULATION EXAMPLE

If a constellation has 4 stars and 3 connecting lines:
Value = (4 × 3) + 4 = 12 + 4 = 16

Calculate the value for each constellation (I, II, III), then concatenate the three values.
Format: [Value I][Value II][Value III] — if any value is single digit, keep it as is (e.g., 5, 12, 7 → "5127")

Constellation I (west): 1 star, 0 connections → (1×0)+1 = 1
Constellation II (center): 5 stars, 0 connections → (5×0)+5 = 5
Constellation III (east): 1 star, 0 connections → (1×0)+1 = 1
Concatenate: 1, 5, 1 → "151"
